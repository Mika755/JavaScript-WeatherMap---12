## Oversikt

Dette prosjektet er en skoleoppgave.  
Målet er å øve på:

- JavaScript, API, Fetch

## Oppgavebeskrivelse
Lag en nettside med et enkelt søkefelt hvor man kan søke på en by i Norge og få opp været til det området. Når man trykker på knappen (Search) eller klikker på «Enter» skal informasjon om byen vises på skjermen.

Krav til siden:
- Input felt man kan søke på
- Søke med både Button og Enter
- Hente informasjon om været fra API
- Dekorere med CSS
